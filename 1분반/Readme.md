- 스터디 당일 전까지 각자 공부 및 정리, 실습을 해오고, 질문 3개 만들어오기

- 각자 가져온 질문들을 서로 공유, 토론하여 해결하기
- 모든 인원의 주간 공부 방법 및 내용 공유
- 개념 원리적인 퀴즈 인당 3개씩 출제하여 퀴즈 풀이 (벌칙은 5등 -> 1등 커피 사주기)
